# CV-GPWR
# 💼 CV Web – Projet Git, GitKraken & GitHub Pages

Projet réalisé dans le cadre d’un exercice pédagogique visant à comprendre le fonctionnement de **Git**, **GitKraken** et **GitHub Pages**, à travers le développement d’un **CV interactif en HTML, CSS et JavaScript**.

---

## 🎯 Objectifs du projet

- Comprendre le fonctionnement de Git (branches, commits, merge, tags)
- Utiliser GitKraken comme interface graphique
- Publier un site web statique avec GitHub Pages
- Développer un CV web interactif

---

## 🛠️ Technologies utilisées

- HTML5
- CSS3
- JavaScript
- Git
- GitKraken
- GitHub Pages

---

## ✨ Fonctionnalités

- 🎯 Filtrage dynamique des compétences
